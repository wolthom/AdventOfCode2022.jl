AdventOfCode2022.jl Tutorial

AdventOfCode2022 is a package that contains solutions and helper functions for the 2022 Advent of Code edition in Julia. This tutorial will walk you through its main functionality and how to use the helpers. Note: This Tutorial assumes that you have a session token from the Advent of Code website.

Tutorial - copy-pasteable version

using AdventOfCode2022
st = ENV["session"]
@run_days 22 st
